## üêõ Troubleshooting

### Auto-Detection Not Working?
- Check that auto-detection is enabled in settings (‚öôÔ∏è)
- Write at least 50 characters of code
- Ensure code has language-specific syntax/keywords
- Try manually selecting language if confidence is low

### Command Palette Not Opening?
- Press `Ctrl+K` (ensure focus is on editor)
- Check browser console for errors
- Verify CommandPalette component is rendered

### Live Preview Issues?
- Ensure you're using a web language (HTML/CSS/React/Vue/SVG)
- Check browser console for errors
- Verify @babel/standalone is loaded for React
- For Vue, ensure proper `<template>` and `<script>` structure
