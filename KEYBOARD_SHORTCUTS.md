## ğŸ†˜ Troubleshooting

### Shortcut Not Working?
1. **Check if command is available** - Some commands require specific contexts
2. **Browser conflicts** - Some browsers override shortcuts (e.g., `Ctrl+W`)
3. **Focus issues** - Ensure editor has focus (click in code area)
4. **Operating system conflicts** - Some OS shortcuts take precedence

### Common Conflicts
- `Ctrl+S` - Browser's save page (CodeRipper downloads instead)
- `Ctrl+W` - Browser's close tab (use with caution)
- `Ctrl+T` - Browser's new tab (may conflict with symbol search)

### Solutions
- Use Command Palette (`Ctrl+K`) as alternative
- Customize shortcuts in settings
- Disable browser shortcuts (if possible)
- Use browser in app mode (`--app` flag)
